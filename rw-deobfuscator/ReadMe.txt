Обратите внимание, что по-умолчанию в исходном коде определены следующие уставки:
	PARTS_NUM = 6 (Количество частей, на которое разобран файл)
	VARIANT_KEY = 23 (Ключ для дешифровки варианта перестановки в теле файла)
	TEMP_BUFFER_SIZE = 8388608 (Макимальный размер зашифрованного файла)

Что это такое?
---------------
Библиотека для инжектирования в тело игры. Предназначена для деобфускации моделей на этапе их загрузки игрой.

Как установить?
------------------
Для ее функционирования должен быть установлен ASI Loader. Библиотеку RWDeobfuscator.asi нужно поместить в папку scripts в корне игры GTA San Andreas. Рекомендуется использовать Silent's ASI Loader 1.3. Ссылка для загрузки: http://www.gtagarage.com/mods/show.php?id=21709
