Обратите внимание, что по-умолчанию в исходном коде определены следующие уставки:
	PARTS_NUM = 6 (Количество частей, на которое разобран файл)
	VARIANT_KEY = 23 (Ключ для дешифровки варианта перестановки в теле файла)

Что это такое?
---------------
Программа запутывает файлы GTA методом перестановки секций. Файл делится на определенное фиксированное количество секций, которые впоследствии случайным образом перемешиваются. Вариант перестановки хранится в файле и модулируется ключом VARIANT_KEY. Таким образом обеспечивается достаточный уровень безопасности и отличные скоростные характеристики. Сложность разбора связана:
	- При изучении файла невозможно установить количество секций в нем.
	- Так как вариант перестановки промодулирован неизвестным исследователю файла числом, извлечь из файла его невозможно.
Разобрать файл можно только грубым перебором и последующим сравнением структуры файла. При PARTS_NUM частей и VARIANT_KEY ключе, исследователю файла постребуется перебрать до (PARTS_NUM! * 0xffffffff) вариантов.

Как пользоваться?
------------------
Программа поставляется вместе с пакетным файлов BuildAll.bat, который автоматически запутывает файлы из папки source и помещает их в папку output.
При необходимости вручную запутать файл используется синтаксис RWObfuscator.exe input_file output_file
