# gta-obfuscation
Obfuscation libraries for protection content of GTA mods

Одна из проблем, встающих перед мододелами GTA заключается в защите разработанного самолично контента. Любые игровые ресурсы, будь то содержимое *.img архивов, или data файлов, легко извлекаются и модифицируются всеми желающими. Для решения этой проблемы и призвана обфускация. Набор из библиотеки(деобфускатора), встраиваемой в процесс игры, и программы-запутывателя(обфускатор) в достаточной степени защитит любой контент внутри игры.
